# PasajeTrak Landing Page

## Overview

This repository contains the landing page for PasajeTrak, a transport expense tracking concept focused on helping users monitor and optimize daily commuting costs.

The page is built as a static website using semantic HTML, Tailwind CSS via CDN, and custom external CSS and JavaScript files.

## Project Goals

- Present PasajeTrak's value proposition clearly.
- Showcase core product features and benefits.
- Provide trust-oriented sections, including Privacy and Terms.
- Deliver a visually consistent, responsive experience across desktop and mobile devices.

## Technology Stack

- HTML5
- Tailwind CSS (CDN)
- Custom CSS
- Vanilla JavaScript
- Google Fonts and Material Symbols

## Project Structure

- `pasajetrak.html`: Main landing page document.
- `assets/css/styles.css`: Custom styles for layout details, animations, card interactions, and visual effects.
- `assets/js/tailwind-config.js`: Tailwind theme extension and design token definitions.
- `assets/js/main.js`: Interactive behavior (mobile menu and reveal-on-scroll logic).

## How to Run Locally

This is a static project and does not require a build step.

1. Clone or download the repository.
2. Open `pasajetrak.html` directly in your browser.
3. For a better development workflow, open the folder in VS Code and use a local static server extension.

## Notes on Styling and Consistency

- The visual design uses a dark surface palette with yellow/gold accents.
- Utility classes from Tailwind are combined with custom classes for specific effects.
- Layout spacing and typography follow a consistent editorial style.

## Maintenance Guidelines

- Keep all inline styling/scripts externalized when possible.
- Preserve naming consistency for section IDs used by navigation links.
- Validate content language consistency (Spanish for user-facing UI text).
- Review accessibility attributes (`alt`, keyboard focus states, and semantic hierarchy) when updating content.
